spitfyre

Spitfyre is a simple application that utilizes the Urban Dictionary API and allows a user to lookup slang terminology or substitute a word with a similar slang term. The results are often hillarious and sometimes inappropriate! You can copy your newly created sentence to your clipboard and send it off in a text message or Google search. This application won first place at General Assembly's February 2017 Hackathon.

JQuery Breakout

JQuery Breakout is a re-creation of a classic Atari arcade game using HTML, Canvas, CSS, and JQuery. This project was conceptualized and created over a one week development sprint and served as my first foray into game physics and HTML audio.

Between Bots

Between Bots is a pair of Twitter bots written in Python with a user interface built on Django and postgreSQL. The unique text is generated using a selectable corpus and a Markov chain algorithm. The bots are semi-sentient in that they catalog content from other users that tweet at them and respond in turn.

Word Twister

Word Twister is a classic word scrambler game. The application runs on the React framework and utilizes a custom built dictionary of words that are randomly scrambled. This project utilizes Javascript fundamentals, DOM manipulation, and core React principles / data flow.

Subtvtle is an Angular application that lets users change the displayed subtitles of an uploaded film on the fly and save the new text to a database. This application uses MongoDB and Mongoose to edit and save user entered data. Feel free to clone or download the project from the repository to explore the functionality.
